# Wiki 知识库 — 文档中心

> 模块：Wiki 知识库 / 空间 + 树形页面 + Markdown + 双向链接 + 知识图谱 + ACL + 工作流发布 + 导出
> 对应 Flyway：V53 · 状态：✅ 前后端已实现并通过 API 验收

Wiki 是平台内置的结构化知识库，支持 Markdown 编写、Obsidian 风格的双向链接与知识图谱、目录/页面级权限控制、发布审批工作流，以及单页/批量导出（含图片）。

## 快速开始（3 步）

1. **建空间**：左侧导航「知识库」→「新建空间」，填名称即可
2. **写页面**：进入空间 → 页面树「新建页面」→ Markdown 编辑器编写 → `Ctrl+S` 保存
3. **发布**：页面右上「发布」（有 `wiki:publish` 权限）或「提交审批」（走工作流）

## 核心概念速览

| 概念 | 说明 |
|------|------|
| **空间（Space）** | 顶级知识库容器，类似 Obsidian 的 Vault 或 Confluence 的 Space |
| **页面（Page）** | 树形组织的 Markdown 文档，可无限层级嵌套 |
| **双向链接** | 在正文用 `[[页面标题]]` 引用其他页面，自动建立反向链接 |
| **反向链接** | 页面侧栏显示「哪些页面引用了本文」 |
| **知识图谱** | 以页面为节点、双向链接为边的可视化关系图 |
| **版本历史** | 每次保存生成快照，可对比与回滚 |
| **ACL** | 页面级访问控制（read/write/delete/publish），覆盖式继承父页 |
| **状态机** | draft（草稿）→ review（审批中）→ published（已发布）→ archived（归档） |
