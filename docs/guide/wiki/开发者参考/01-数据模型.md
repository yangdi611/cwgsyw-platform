# 数据模型

V53 迁移创建 5 张表。附件复用现有 `shared_file` 表（`source_type='wiki_page'`），不新建附件表。

## ER 关系

```
wiki_space (1) ──< (N) wiki_page (自引用树 parent_id)
                          │
                          ├──< (N) wiki_page_version   版本快照
                          ├──< (N) wiki_page_acl        访问控制
                          └──< (N) wiki_backlink         出链(from_page_id)
shared_file (source_type='wiki_page', source_id=page.id)  附件
```

## wiki_space — 空间

| 列 | 类型 | 说明 |
|----|------|------|
| id | BIGSERIAL | PK |
| tenant_id | VARCHAR(64) | 多租户，默认 `default` |
| name | VARCHAR(255) | 名称 |
| description | TEXT | 描述 |
| created_by/at, updated_by/at | | 审计列 |
| is_deleted, deleted_at, deleted_by | | `@TableLogic` 软删 |

## wiki_page — 页面

| 列 | 类型 | 说明 |
|----|------|------|
| id | BIGSERIAL | PK |
| tenant_id | VARCHAR(64) | 租户 |
| space_id | BIGINT | FK → wiki_space |
| parent_id | BIGINT | 自引用，null=根 |
| slug | VARCHAR(255) | URL 友好名，同级唯一 |
| title | VARCHAR(500) | 标题 |
| content | TEXT | Markdown 正文 |
| status | VARCHAR(32) | draft/review/published/archived |
| current_version | INT | 当前版本号 |
| process_instance_id | VARCHAR(255) | Flowable 实例（审批中） |
| sort_order | INT | 同级排序 |
| acl_inherited | BOOLEAN | true=继承父级 ACL |
| 审计 + 软删列 | | |

索引：`idx_wiki_page_space`、`idx_wiki_page_parent`（均 `WHERE NOT is_deleted`）。

## wiki_page_version — 版本快照

| 列 | 说明 |
|----|------|
| id, tenant_id, page_id | |
| version | 版本号 |
| title, content | 该版本全量快照 |
| comment | 版本备注 |
| created_by, created_at | |

**无软删**，版本永久保留。索引 `(page_id, version DESC)`。

## wiki_backlink — 反向链接

| 列 | 说明 |
|----|------|
| id, tenant_id | |
| from_page_id | 含 `[[ ]]` 的来源页 |
| to_page_id | 被引用的目标页 |
| created_at | |

唯一约束 `(from_page_id, to_page_id)`，索引 `to_page_id` 和 `from_page_id`。保存时按 from 清空重建。

## wiki_page_acl — 访问控制

| 列 | 说明 |
|----|------|
| id, tenant_id, page_id | |
| subject_type | role/group/user |
| subject_id | 对应 ID |
| permissions | JSONB `["read","write","delete","publish"]` |
| 审计 + 软删列 | |

`autoResultMap=true` + `JacksonTypeHandler` 处理 JSONB。

## 关键约定

- 时间列用 `TIMESTAMP`（非 TIMESTAMPTZ）
- 软删表通过 `@TableLogic`；**删除用 `deleteById`**（手动 set isDeleted 会被忽略）
- `permissions` JSONB 查询必须用 `LambdaQueryWrapper`（禁 `@Select`，否则 TypeHandler 失效读到 null）
- 全文索引：**未建 GIN**（simple 配置对中文无效），搜索走 ILIKE
