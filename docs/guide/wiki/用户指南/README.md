# 用户指南

面向所有 Wiki 使用者，覆盖日常编写、组织、检索与导出。

## 章节

1. [空间与页面管理](./01-空间与页面.md) — 创建空间、新建/移动/删除页面、页面树
2. [Markdown 编辑器](./02-Markdown编辑器.md) — 分栏编辑、图片上传、自动保存、快捷键
3. [双向链接与知识图谱](./03-双向链接与知识图谱.md) — `[[ ]]` 链接、反向链接、图谱视图
4. [版本历史](./04-版本历史.md) — 查看历史、对比、回滚
5. [搜索](./05-搜索.md) — 全文搜索与定位
6. [发布与导出](./06-发布与导出.md) — 发布/提交审批、单页与批量导出

---

## 整体使用流程

```
建空间 → 建页面（draft）→ 编辑 Markdown → 保存（自动生成版本）
   → 用 [[ ]] 互链 → 发布 / 提交审批 → published
   → 需要时导出 .md / .zip
```
