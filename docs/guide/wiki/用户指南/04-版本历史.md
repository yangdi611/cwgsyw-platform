# 版本历史

每次保存页面都会生成一个全量快照，永久保留（版本记录不会被删除）。

## 查看版本

页面阅读视图右侧 **版本历史** 面板（可折叠），列出该页所有版本：

| 列 | 说明 |
|----|------|
| 版本号 | 自增整数，最新在前 |
| 时间 | 保存时间 |
| 操作人 | 该版本的保存者 |
| 备注 | 保存时填写的版本说明（如「首次编写」「回滚自版本 3」） |

## 回滚

1. 在版本列表中选择目标历史版本
2. 点击 **回滚**
3. 确认后，系统以该历史版本的标题+正文创建一个**新版本**（不是覆盖历史）
4. 版本备注自动标记为「回滚自版本 N」

> 回滚是「前进式」的：它生成新版本而非删除中间版本，历史链路完整可追溯。

## 版本与编辑的关系

- 自动保存（30s）和手动保存（Ctrl+S）都会产生新版本
- 频繁自动保存会产生较多版本，这是预期行为（内部团队体量无压力）
- 版本内容存于 `wiki_page_version` 表（全量快照）
