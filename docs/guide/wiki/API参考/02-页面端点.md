# 页面端点

## GET /api/wiki/pages/{id}

页面详情（含正文）。权限 `wiki:read` + ACL `read`。

**响应**
```json
{
  "code": 200,
  "data": {
    "id": 1, "space_id": 1, "parent_id": null,
    "title": "部署手册", "slug": "...", "content": "# 部署手册\n...",
    "status": "published", "current_version": 2, "acl_custom": false,
    "updated_at": "2026-06-25T16:42:00", "updated_by_name": "杨迪", "backlink_count": 0
  }
}
```

---

## POST /api/wiki/pages

新建页面。权限 `wiki:create`（若指定 `parent_id` 还需父页 ACL `write`）。

**请求**
```json
{ "space_id": 1, "parent_id": null, "title": "部署手册" }
```
- `parent_id` 为 `null` → 根页面
- 新页面初始 `status=draft`，`current_version=1`，自动生成 `slug`

---

## PUT /api/wiki/pages/{id}

保存页面正文。权限 `wiki:update` + ACL `write`。

**请求**
```json
{ "title": "部署手册", "content": "# 部署手册\n参见 [[网络拓扑]]", "comment": "首次编写" }
```
- 版本号 +1，生成版本快照
- 解析 `[[ ]]` 重建反向链接
- `status=archived` 的页面拒绝保存

---

## DELETE /api/wiki/pages/{id}

删除页面。权限 `wiki:delete` + ACL `delete`。

- **级联软删除**：连同所有子孙页面一起删除
- 通过 `deleteById`（`@TableLogic`）逻辑删除

---

## POST /api/wiki/pages/{id}/move

移动/排序。权限 `wiki:update` + ACL `write`。

**请求**
```json
{ "parent_id": null, "sort_order": 5 }
```
- 防环：不能移动到自身子孙下，否则报错

---

## POST /api/wiki/pages/{id}/submit

提交审批。权限 `wiki:update`。

- 仅 `draft` 可提交，否则报错
- 启动 Flowable 流程 `wiki_publish`，`status=review`

---

## POST /api/wiki/pages/{id}/publish

直接发布。权限 `wiki:publish`。`status=published`，通知创建者。

---

## GET /api/wiki/pages/{id}/versions

版本列表（最新在前）。权限 `wiki:read`。

**响应**
```json
{ "code": 200, "data": [
  { "version": 2, "title": "部署手册", "comment": "首次编写", "created_by_name": "杨迪", "created_at": "..." },
  { "version": 1, "title": "部署手册", "comment": "初始版本", "created_by_name": "杨迪", "created_at": "..." }
]}
```

---

## POST /api/wiki/pages/{id}/revert/{version}

回滚到指定版本。权限 `wiki:update` + ACL `write`。以历史版本内容生成**新版本**。

---

## GET /api/wiki/pages/{id}/export

单页导出。权限 `wiki:read` + ACL `read`。

- 无图片 → `Content-Type: text/markdown`，下载 `.md`
- 含图片 → `Content-Type: application/zip`，下载含 `images/` 的 zip

---

## GET /api/wiki/pages/{id}/backlinks

反向链接（引用本页的页面）。权限 `wiki:read`。

**响应**
```json
{ "code": 200, "data": [ { "page_id": 1, "title": "部署手册", "space_id": 1 } ] }
```
