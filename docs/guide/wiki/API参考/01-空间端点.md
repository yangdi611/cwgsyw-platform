# 空间端点

## GET /api/wiki/spaces

列出当前租户所有空间。权限 `wiki:read`。

**响应**
```json
{
  "code": 200, "message": "success",
  "data": [
    { "id": 1, "name": "运维知识库", "description": "...",
      "page_count": 3, "updated_at": "2026-06-25T16:41:51", "created_by_name": "杨迪" }
  ]
}
```

---

## POST /api/wiki/spaces

创建空间。权限 `wiki:create`。

**请求**
```json
{ "name": "运维知识库", "description": "API验收测试空间" }
```

**响应** — 返回创建的空间（`page_count=0`）。

```bash
curl -X POST http://localhost:8081/api/wiki/spaces \
  -H "Authorization: Bearer $T" -H "Content-Type: application/json" \
  -d '{"name":"运维知识库","description":"测试"}'
```

---

## PUT /api/wiki/spaces/{id}

更新空间名称/描述。权限 `wiki:update`。请求体同 POST。

---

## DELETE /api/wiki/spaces/{id}

删除空间。权限 `wiki:delete`。

- **约束**：空间内还有页面时返回 `409`，message=`空间非空，请先删除全部页面`
- 空间为空时返回 `200`，软删除

---

## GET /api/wiki/spaces/{id}/tree

返回空间的页面树（递归嵌套）。权限 `wiki:read`。

**响应**
```json
{
  "code": 200, "data": [
    { "id": 1, "title": "部署手册", "slug": "...", "status": "published",
      "sort_order": 0, "space_id": 1,
      "children": [
        { "id": 3, "title": "子章节", "status": "draft", "children": [] }
      ]
    },
    { "id": 2, "title": "网络拓扑", "status": "draft", "children": [] }
  ]
}
```

---

## GET /api/wiki/spaces/{id}/graph

知识图谱数据。权限 `wiki:read`。仅含 `published` 页面，仅含两端均已发布的边。

**响应**
```json
{
  "code": 200,
  "data": {
    "nodes": [ { "id": "1", "title": "部署手册", "status": "published" } ],
    "edges": [ { "source": "1", "target": "5" } ]
  }
}
```

---

## GET /api/wiki/spaces/{id}/export

批量导出整个空间为 zip（流式）。权限 `wiki:read`。

- 响应头：`Content-Type: application/zip`，`Content-Disposition: attachment; filename="<空间名>.zip"`
- zip 内保留页面树目录结构，图片归集到 `images/` 并全局去重
- 文件名 UTF-8 编码

```bash
curl http://localhost:8081/api/wiki/spaces/1/export -H "Authorization: Bearer $T" -o space.zip
```
