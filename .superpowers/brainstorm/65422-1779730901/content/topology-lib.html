拓扑树渲染方案

选择 CI 关联关系图的渲染方式。项目技术栈：Next.js 15 / React 19，无现有图形库。

各方案示意（以"主机 web-01 → 应用 web-app → MySQL db-01"为例）
A — React Flow
可拖拽节点，内置缩放/平移，支持自定义节点样式。需安装 react-flow 包（~2MB）
B — Cytoscape.js
专业图分析库，内置多种布局算法（力导向/层次/树形），适合大量节点。包体较大（~3MB）
C — 纯 SVG（自实现）
零依赖，完全自控，简单层级布局。无交互拖拽，图形复杂时布局算法需自行实现
A

React Flow（推荐）

React 生态最成熟的流程图库，可拖拽节点、缩放、自定义节点/边样式，API 友好。适合 Next.js 项目，社区活跃，包体约 2MB。

B

Cytoscape.js

专业图分析库，内置力导向、层次、树形等布局算法，适合节点数量大（50+）的复杂拓扑。需要封装 React wrapper，包体约 3MB。

C

纯 SVG 自实现

零依赖，完全控制样式和布局。适合简单的树形结构，但缺少拖拽/缩放等交互，复杂图形需要自行实现布局算法。
