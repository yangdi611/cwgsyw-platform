# 如何提交反馈

欢迎来到 Bug 反馈与建议空间。这里是大家共同完善平台的地方，**所有同事都可以在此新建页面、记录问题和想法**。

## 三种反馈类型

根据你要提交的内容，参考对应模板新建页面：

- **[[【模板】Bug 报告]]** — 平台出现错误、异常、用不了的情况
- **[[【模板】优化建议]]** — 功能能用，但体验上可以更好
- **[[【模板】新功能需求]]** — 希望平台增加新的能力

## 提交步骤

1. 打开对应模板页面，复制里面的内容
2. 在本空间新建一个页面（左侧目录树点 + 号）
3. 粘贴模板内容，按提示填写
4. 页面标题建议写清楚，例如「Bug：设备密码复制后无法粘贴」
5. 保存即可

## 小提示

- 标题尽量具体，方便他人快速了解
- Bug 报告请尽量写清**复现步骤**，这对定位问题最有帮助
- 模板页面本身是只读的（防止被误改），请新建自己的页面填写
- 大家可以互相查看、补充评论，共同跟进

感谢你的反馈，每一条都会帮助平台变得更好。
