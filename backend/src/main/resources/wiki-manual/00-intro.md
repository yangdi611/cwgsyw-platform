# 平台简介与快速上手

欢迎使用 IT 基础设施运维运营管理平台。本手册介绍平台各功能模块的用途与使用方法，帮助你快速上手。

## 平台能做什么

本平台帮助运维团队集中管理 IT 基础设施与日常运维工作，主要功能包括：

- **[[CMDB 配置管理]]** — 管理服务器、应用等配置项及其关联关系，是平台的数据基础
- **[[设备密码库]]** — 集中、加密保管设备登录凭据
- **[[变更文档]]** — 规范化的变更申请、审批与归档
- **[[日报系统]]** — 团队日报填报与审批
- **[[共享文件库]]** — 团队文件的分类存储与共享
- **[[Wiki 知识库使用说明|Wiki 知识库]]** — 团队知识沉淀（本手册即在此）
- **[[系统管理]]** — 用户、权限、系统配置与数据备份（管理员）

## 如何登录

1. 打开平台地址，输入用户名和密码
2. 登录后进入[[工作台]]首页
3. 左侧导航栏列出你有权限访问的所有模块

## 导航说明

左侧导航栏按模块分组。你能看到哪些菜单，取决于管理员分配给你的角色权限。如果某个功能你看不到，说明你当前没有对应权限，可联系管理员开通。

页面右上角是通知铃铛（有未读消息时显示红点）和个人菜单（退出登录在此）。

## 角色与权限

平台用「角色」控制每个人能做什么。常见角色：

| 角色 | 能做什么 |
|------|---------|
| 超级管理员 | 全平台所有功能 |
| 管理员 | 本租户所有功能 |
| 组长 | 本组管理 + 审批 |
| 组员 | 基本读写 |
| 只读 | 仅查看 |

具体权限以管理员实际分配为准。详见 [[用户与权限管理]]。

## 关于本手册

本手册由系统统一维护，随平台版本更新自动同步，普通用户为只读浏览。每个模块的详细用法见对应章节。
