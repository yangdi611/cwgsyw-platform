# v0.21.0 — 2026年6月28日

本次更新聚焦**机柜视图的视觉重做**，把原先的色块布局换成更贴近真实机架的样子，并补充了硬件信息悬停浮卡。

## 优化

### 机柜视图重做

机柜实例详情页的「机柜视图」由原先的纯色方块重做为接近真实数据中心机柜的样子，一屏即可完整呈现整座机柜：

- **真实导轨** — 左右立柱画出安装孔，U 编号两侧标注；空 U 位清晰可辨，已装与空置一眼区分
- **按设备类型呈现** — 服务器、网络设备、电源、存储等画出各自的面板样式，左侧带一条模型配色
- **状态指示灯** — 每台设备一颗状态灯：运行（绿）、告警（琥珀）、异常（红）、未知（灰）
- **深色金属质感** — 机架本体采用深色金属观感，让配色与状态灯更突出
- **一屏自适应** — 整座机架自动缩放，再高的机柜也能完整显示且不失真

### 硬件信息悬停浮卡

鼠标悬停机架上任一设备，即跟随显示一张信息卡片：设备名称、状态、型号、U 位区间、内网 IP、资产编号、序列号。点击设备仍可进入其详情页，并支持键盘操作。

## 修复

- **实例详情加载更可靠** — 实例详情页此前会把网络波动、会话过期等各种问题统一显示为「实例不存在」。现在能区分确实不存在与其他临时错误，临时错误会自动重试并提供手动重试入口
- **知识库编辑器文字不可见** — 修复知识库页面编辑区文字显示为白色、看不见的问题
