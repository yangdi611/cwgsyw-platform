# v0.21.0 — 2026年6月28日

本次更新聚焦 **CMDB 机柜视图（2D Rack Elevation）的视觉重做**，把原先的色块布局换成更贴近真实机架的矢量图，并补充了硬件信息悬停浮卡。

## 改进

### 机柜视图重做为矢量机架

CMDB 机柜实例详情页的「机柜视图」由原先的纯色方块布局重做为 **SVG 矢量机架**，更接近真实数据中心机柜的样子，且一屏可完整呈现 42U：

- **导轨孔条** — 左右两条立柱画出 cage-nut 方孔（每 U 三孔），U 编号在两侧同时标注；空 U 位露出暗色内腔，已装/空置一眼可辨
- **按设备类型渲染面板** — 不同模型画不同面板纹理：服务器=硬盘托架、网络设备=端口阵列、电源/PDU=插座、存储=磁盘点阵、其他=散热栅格；面板左侧一条模型配色条
- **状态指示灯** — 每块面板右侧一颗状态 LED（运行=绿 / 告警=琥珀 / 异常=红 / 未知=灰），带呼吸光晕动画
- **深色金属质感** — 机架本体采用深色金属拉丝渐变，让彩色配色条与状态灯更突出，与真实硬件观感一致
- **一屏自适应** — 整张机架用 SVG `viewBox` 自适应缩放（高度上限 70vh），42U 也能完整塞进一屏且缩放不失真

### 硬件信息悬停浮卡

鼠标悬停机架上任一设备，跟随显示一张信息浮卡：

- 设备名称 + 状态徽标（带状态灯）
- 型号、U 位区间、**内网 IP**、资产编号、**序列号（SN）**
- 点击设备面板仍跳转至该实例详情页；支持键盘 Tab 聚焦 + Enter 进入

### 其他修复

- **实例详情加载容错** — 实例详情页原先用 `catch` 吞掉所有请求错误统一显示「实例不存在」，会把网络抖动/会话过期等误报为不存在。现区分 404（确实不存在）与其他错误（显示错误码 + 重试按钮），并对非 404 错误自动重试
- **知识库编辑器字体不可见** — 修复 Wiki 页面编辑区文字显示为白色看不见的问题（第三方编辑器库的 `-webkit-text-fill-color` 被置为透明，已在编辑区作用域内显式设回前景色）

## 技术细节

- **后端**（只读、无迁移）：`findRackMembers` SQL 投影增取 `attrs->>'inner_ip'` 与 `attrs->>'sn'`，`RackMemberRow` / `RackDeviceVO` 增加 `innerIp`、`sn` 字段并由 `RackLayoutService` 透传；全 camelCase，未引入 `@JsonNaming`（遵循 v0.20.0 起的全局命名规范）
- **前端**：`RackElevationView.tsx` 由 `absolute` 定位的 div 块重写为响应式 SVG，含金属渐变 `defs`、cage-nut 孔条、按类型的面板纹理组件、跟随鼠标的悬停浮卡；保留顶部布局告警条与「未定位设备」列表
- **验证**：tsc 类型检查 + 后端 Maven 编译 + 前端 `next build` 全过；Playwright 实测机架渲染、悬停浮卡（IP/SN）、设备跳转均正常
