# v0.14.0 — 2026年6月26日

本次更新为平台引入了在线知识库手册，并改善了版本管理。

## 新增

### 平台使用手册（在线）
平台现在内置一份完整的在线使用手册，覆盖所有功能模块——CMDB 配置管理、设备密码库、日报、变更文档、共享文件库、Wiki、系统管理等。你可以在「Wiki 知识库」中找到「平台使用手册」空间，随时查阅每个功能的用途和操作方法。

- 手册按模块分章节，目录清晰
- 章节之间有相互引用链接，方便跳转
- 内容由系统统一维护，会随平台更新保持最新

### 版本号展示
左下角现在显示当前平台版本号，方便你确认正在使用的版本。

## 改善

- 平台发布流程优化，新功能能更快、更稳定地上线
