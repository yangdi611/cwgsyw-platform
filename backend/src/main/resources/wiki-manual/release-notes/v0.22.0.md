# v0.22.0 — 2026年6月28日

本次更新重做了 **CMDB 概览的模型浏览方式**，并完善了模型分类的整理能力，同时修复了若干交互与显示问题。

## 新增功能

### 概览页模型分类文件夹

CMDB 概览页的「各类模型」从平铺所有模型改为**分类文件夹**：

- 默认只展示分类，每个分类卡片用其中模型的配色拼成预览图，并显示模型数量与实例总数，一眼掌握全局
- 点击分类即弹出该类下的所有模型，点选任一模型进入对应的实例列表
- 模型再多也不会铺满整屏，浏览更清爽

### 移动模型到其他分类

模型卡片的操作菜单新增「移动到分类」，可直接把模型归入其他分类，并支持一键撤销。

## 优化

### 模型管理页合并为「模型目录」

原先分散的「模型管理」与「模型分类」合并为统一的「模型目录」：分类以可展开的形式呈现，展开后即见该类下的模型，新建分类与新建模型入口并排放置，整理结构更顺手。

### 内置模型分类归位

修正了部分内置模型的初始分类：主机、存储、SAN 交换机、网络交换机统一归入「硬件设备」。同时移除了归位后空置的「主机管理」与「网络设备」两个内置分类。

## 修复

- **模型操作菜单** — 修复「打开设置」点击无反应，以及鼠标移到「移动到分类」时页面崩溃的问题
- **关联定义不显示** — 修复「关联定义」中已有定义却显示为空的问题；加载失败时不再静默显示为空，而是明确提示原因并提供重试
