# v0.15.0 — 2026年6月26日

本次更新完善了知识库的组织方式，并新增了发布说明与反馈渠道。

## 新增

### Release Notes 发布说明
平台新增「Release Notes」空间，每次版本更新都会在这里发布一篇说明，告诉你这一版**修复、新增、改善了哪些功能**。你现在看到的就是发布说明。

### Bug 反馈与建议
新增「Bug 反馈与建议」空间，所有同事都可以在这里提交：

- **Bug 报告** — 发现问题时填写
- **优化建议** — 觉得哪里可以更好用
- **新功能需求** — 希望平台增加什么能力

空间里预置了三种模板，照着填即可。大家一起把平台打磨得更好用。

## 改善

### 知识库分层更清晰
「Wiki 知识库」首页现在分为两层：

- **官方手册** 区（置顶）：平台使用手册、Release Notes、Bug 反馈，由系统统一维护
- **团队空间** 区：你和团队自己创建的知识空间

两层之间有清晰的分隔，官方内容一目了然，团队空间也不再混杂其中。
