# v0.18.0 — 2026年6月27日

本次更新带来了**全局搜索**，一处入口搜遍平台所有核心资源。

## 新功能

### 全局搜索（⌘K 命令面板）

顶栏中间的搜索框现在真正可用了。点击它，或在任意页面按 **⌘K / Ctrl+K**，即可呼出一个 Spotlight 风格的命令面板，一次搜遍六类资源：

- **配置项 (CI)** — 按名称或属性（IP、序列号、环境等）
- **共享文件** — 按文件名
- **变更单** — 按标题或单号
- **设备** — 按名称、IP、分类
- **用户** — 按用户名或姓名
- **知识库** — 按页面标题或正文内容

搜到结果直接回车或点击，一键跳转到对应详情页。

具体体验点：

- **干净起手** — 打开后只有一个搜索框，输入后结果才出现，不堆砌空画布
- **按权限收敛** — 只会搜到你有权限查看的资源类型，没权限的类别自动跳过
- **关键词高亮** — 命中的关键词在结果里加粗醒目
- **知识库聚光灯** — 知识库正文命中时，会把关键词所在的片段拉到中间、两侧文字渐隐，让你一眼看到匹配位置；其余结构化信息（IP、单号等）则清爽右对齐
- **分组展示** — 结果按资源类型分组，层次清晰
