# v0.17.0 — 2026年6月27日

本次更新改善了顶部导航栏的位置指示，让你随时清楚自己在平台的哪个位置。

## 改善

### 顶部面包屑导航
顶栏左上角原来只显示一个「当前位置 / 模块名」，现在升级为完整的**面包屑路径**，从所属模块一路标到当前页面。

举几个例子：

- `CMDB › 实例管理 › web-prod-01`
- `变更文档 › 文档列表 › 数据库迁移变更单`
- `资源管理 › 设备密码库 › 核心交换机-01`
- `知识库 › 运维知识库 › 部署手册`

具体改善点：

- **层级一目了然** — 不论翻到多深的页面，都能看清完整路径
- **路径可点跳转** — 面包屑中间的层级（如模块名、知识空间名）都能点击，一键回到上级
- **显示真实名称** — 进入某台主机、某篇文档、某个 IP 池时，路径末尾直接显示它的真实名字，而不是一串编号；名称加载中时会先以占位符过渡，加载完自动补上
- **风格统一克制** — 模块图标采用品牌蓝点缀，其余层级为安静的灰阶，当前页面加粗显示，与平台整体视觉保持一致
