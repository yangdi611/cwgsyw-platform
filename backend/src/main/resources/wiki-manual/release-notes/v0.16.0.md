# v0.16.0 — 2026年6月26日

本次更新围绕知识库编辑体验和导航栏交互做了多项改善。

## 新增

### 知识库支持插入图片
编辑文档时，现在可以直接插入图片：

- **粘贴截图** — 复制剪贴板里的图片后，在编辑器里直接粘贴即可
- **选择本地图片** — 点工具栏的图片按钮，从电脑里选一张图

支持 PNG / JPG / GIF / WEBP / SVG 格式，单张不超过 5MB。图片会自动上传保存，并在文档里按合适尺寸显示。

### 侧边导航栏可收起
点击导航栏顶部的收起按钮，导航栏会收窄成只显示图标，给主内容区腾出更多空间。收起状态下把鼠标移到某个图标上，右侧会平滑弹出对应的二级菜单。下次打开平台会记住你的选择。

## 改善

### 知识空间卡片整张可点
在知识库首页，现在点击空间卡片的任意位置即可进入该空间，不必非得点「进入」按钮。

### 编辑器风格统一
文档编辑页的背景和配色与阅读页保持一致，不再有突兀的色块；工具栏图标也更大、更清晰。

### 删除页面确认方式统一
删除知识库页面时，改为弹出与平台其他地方一致的确认对话框，不再使用浏览器原生提示，风格更统一。
