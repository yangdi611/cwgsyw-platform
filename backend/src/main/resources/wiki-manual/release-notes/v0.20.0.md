# v0.20.0 — 2026年6月28日

本次更新补全了配置管理的多项录入与维护能力：结构化表格字段、批量导入、实例克隆、端口级连接，以及设备装入机柜。

## 新增功能

### 表格类型字段

属性新增「表格」类型，用于记录一个实例下的多行结构化数据，例如主机的多块网卡、多个文件系统，或交换机的多个端口。可在模型管理中定义每张表的列，并提供常见结构的快捷模板；在实例详情页以表格形式逐行增删与编辑。

### JSON 批量导入

实例导入除 CSV 外新增 JSON 格式，可表达 CSV 无法承载的多值嵌套数据。提供三种导入模式：

- **合并** — 只更新本次提供的字段
- **字段替换** — 出现的字段即覆盖
- **基线重导** — 按整批数据对齐，自动处理新增、更新与移除，适合定期采集脚本无脑重导

支持指定唯一字段判断新增还是更新，单条失败不影响其余，结束后汇总成功与失败明细。

### 实例克隆

实例列表新增「克隆」，一键复制现有实例的全部字段并自动重命名，省去重复录入。

### 端口级连接

新增端口级连接能力，可精确描述设备之间的物理与逻辑链路，例如网口连交换机口、存储连主机。拥有表格字段的设备详情页出现「端口连接」卡片，可视化新建与解除连接，建立的连接会自动反映到拓扑图中。

### 设备装入机柜

设备详情页新增「所在机柜」，可选择机柜并填写起止 U 位，填好后即在机柜视图中正确定位。

## 优化

- 统一了全平台接口的字段命名规范，使前后端字段保持一致，减少因命名不一致引发的显示异常
- 修复机柜视图中设备链接异常、已填 U 位仍显示「未定位」的问题
