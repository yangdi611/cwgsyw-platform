# v0.27.1 — 2026年7月2日

本次为修复性小版本，集中修正全平台下拉筛选/表单中「选中后显示乱码般的英文代码」的问题。

## 修复

- **下拉框选中后显示英文代码而非中文名称** — 修复了全平台近 60 处下拉筛选和表单选择框的显示问题。此前选中某个选项后，框内可能显示成内部代码（如 `gpu_server`、`__all__`、`1:n`）而不是对应的中文名称（如"GPU服务器"、"全部模型"、"一对多"），选项列表本身是对的，只是选中后回显错了。涉及模块包括：
  - CMDB（变更历史筛选、模型管理、关联定义、机柜装配、端口连接、实例编辑、告警中心、2D 分组视图、影响分析）
  - 运维日历（临时任务、周期规则、执行结果提交、节假日、报表模板、素材归集）
  - 工作流（模板配置、流程绑定）
  - 系统管理（系统配置、审计日志、变更文档模板）
  - 日报、IP 地址池管理、报表导出
- 本次修复不改变任何下拉框的选项内容和数据存储方式，只修正选中后的显示文案，属于纯前端展示层修复。
