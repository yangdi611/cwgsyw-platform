# v0.26.0 — 2026年6月30日

本次更新将「运维日历」模块补齐到设计规格终态：新增统计复盘、模板管理、素材归集导出、节假日相对触发与中国法定节假日一键导入，并补全了管理入口与月历节假日标记。

## 新增

- **统计与复盘面板**（`/ops-calendar/stats`）— 完成率、逾期率、任务类型分布、状态分布、每日趋势、负责人负载 Top 10、逾期排行 Top 10。后端 `GET /api/ops-calendar/stats`。
- **模板管理**（`/ops-calendar/templates`）— 通知模板 / SOP 检查项模板的增删改查；周期规则可关联检查项模板，生成任务时自动复制检查项。后端 `GET/POST/PUT/DELETE /api/ops-calendar/templates`。
- **报表素材归集导出**（`/ops-calendar/materials`）— 按季度 / 半年归集任务执行情况与关联对象（日报、CI、告警、变更文档），支持一键导出 Excel（概览 + 任务明细两个 Sheet）。后端 `GET /api/ops-calendar/report-materials/export`。
- **节假日相对触发（holiday_relative）+ 工作日推算** — 周期规则新增「节前 / 节后第 N 个工作日」触发类型；季度 / 半年规则支持「按工作日偏移」（季末前 N 个工作日，自动跳过周末与节假日）。
- **中国法定节假日一键导入** — 节假日历页新增「导入 2026 法定节假日」按钮，内置元旦 / 春节 / 清明 / 劳动 / 端午 / 中秋 / 国庆 7 个节假日（含调休补班日），幂等导入。导入后可按国务院最终公告调整。
- **节假日删除** — 节假日历页支持删除条目。

## 优化

- **运维日历管理入口补全** — 主日历页顶部新增「排班 / 节假日 / 模板 / 统计 / 素材」入口按钮（按权限显示），此前这些页面只能手敲 URL 进入。
- **月历 / 周历标记节假日** — 月视图与周视图的日期格现在直接显示节假日名称（红色标记），无需点开单日弹窗即可一眼识别国庆、春节等。

## 说明

- 中国法定节假日内置数据为运营估算值，导入后请在节假日历页按国务院当年公告核对调整。
- 至此运维日历模块覆盖 spec 全部交付项（任务闭环、周期规则、排班、节假日、通知、审计、统计、模板、素材归集、节假日自动化），仅外部节假日 API 自动同步留待后续。
