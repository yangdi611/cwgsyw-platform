# v0.26.0 — 2026年6月30日

本次更新把「运维日历」模块补齐到完整形态：在原有的任务闭环之上，新增统计复盘、模板管理、素材归集导出，并让节假日真正融入日历——支持节前节后排期、中国法定节假日一键导入，月历周历上也能直接看到节假日。

## 新增

- **统计与复盘面板** — 一页看清一段时间内的运维全貌：完成率、逾期率、任务类型分布、状态分布、每日趋势，以及负责人负载排行和逾期排行 Top 10。月底复盘、季度汇报时直接取数，不用再手工统计。
- **模板管理** — 可维护通知模板和检查项（SOP）模板。周期规则关联检查项模板后，生成的任务会自动带上这套检查清单，确保同类工作每次都按统一标准执行。
- **报表素材归集导出** — 按季度或半年一键归集任务执行情况和关联对象（日报、配置项、告警、变更文档），并可导出为 Excel 表格，包含「概览」和「任务明细」两张表，为季报、半年报直接提供素材底稿。
- **节假日相对排期** — 周期规则新增「节前 / 节后第 N 个工作日」触发方式；季度、半年类规则也支持「按工作日推算」，比如季末前 N 个工作日，会自动跳过周末和节假日。报税、归档、核查这类卡着节假日节点的工作，从此能精准落在正确的那一天。
- **中国法定节假日一键导入** — 节假日历新增「导入 2026 法定节假日」按钮，内置元旦、春节、清明、劳动、端午、中秋、国庆 7 个节假日（含调休补班日），重复导入也不会产生重复条目。导入后可按国务院当年正式公告再做微调。
- **节假日删除** — 节假日历支持删除已有条目，便于按实际安排维护。

## 优化

- **管理入口补全** — 主日历页顶部新增「排班 / 节假日 / 模板 / 统计 / 素材」入口按钮（按权限显示），相关页面不再需要手动输入地址才能进入。
- **月历 / 周历标记节假日** — 月视图和周视图的日期格现在直接以红色标出节假日名称，国庆、春节等一眼可辨，不必逐天点开查看。

## 说明

- 内置的中国法定节假日为运营估算值，导入后请对照国务院当年公告核对调整。
- 至此「运维日历」覆盖全部规划能力：任务闭环、周期规则、排班、节假日、通知提醒、操作留痕、统计复盘、模板、素材归集与节假日排期。后续仅计划接入外部节假日数据的自动同步。
