# 共享文件库

共享文件库是团队的文件中心，支持文件夹树、上传下载、在线预览、按文件夹设置访问权限，以及变更文档自动归档。

## 浏览文件

进入 **文件管理** 页：

- 左侧是文件夹树，点击文件夹查看其中的文件
- 右侧表格列出文件，支持关键词搜索、分页
- 底部可展开操作日志面板

## 文件夹管理

- **新建文件夹**：在当前位置创建子文件夹
- **删除文件夹**：仅当文件夹为空（无文件、无子文件夹）时可删

## 上传文件

1. 进入目标文件夹
2. 点"上传"，选择文件
3. 上传后文件归入当前文件夹
4. Word 文档会自动生成预览

## 下载与预览

- **下载**：生成临时下载链接（有效期 5 分钟）
- **预览**：在线预览（有效期 30 分钟）
  - PDF 直接在浏览器查看
  - Word（docx）在线渲染
  - Excel 表格在线展示

## 文件夹访问权限（ACL）

可对文件夹设置细粒度的访问权限，控制谁能读/写/改/删：

- **继承（默认）**：跟随父文件夹的权限
- **自定义**：本文件夹用独立权限，可比父级更严或更松

权限主体可以是**角色**、**用户组**或**具体人员**，权限动词包括：

| 动词 | 含义 |
|------|------|
| read | 浏览查看 |
| write | 上传、建子文件夹 |
| update | 重命名 |
| delete | 删文件、删空文件夹 |

设置 ACL 需要 `shared_file:manage_acl` 权限。设置了自定义权限的文件夹会显示锁标记。管理员始终可访问。

## 变更文档归档

[[变更文档]]审批通过后，会自动归档到本文件库（「变更文档 / 年 / 月」目录），无需手动上传。

## 权限

| 权限 | 能做什么 |
|------|---------|
| `shared_file:read` | 浏览、下载 |
| `shared_file:upload` | 上传 |
| `shared_file:delete` | 删除 |
| `shared_file:manage_acl` | 配置文件夹权限 |

相关：[[变更文档]]、[[用户与权限管理]]
