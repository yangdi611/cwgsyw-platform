# 日报系统

日报系统用于团队成员每日填报工作内容，需要审批的日报会进入工作流，最终生成月度/季度统计。

## 填报日报

进入 **日报 → 新建**：

1. 选择日期（默认今天）
2. 填写工作内容（支持 Markdown）
3. 可选关联多个 CMDB 配置项（如"今天维护了哪些服务器"）
4. 选择类别（日常运维 / 项目实施 / 故障处理 / 其他）
5. 保存为草稿 / 提交审批

组长以上角色可以直接发布，不走审批；组员提交后进入审批流程。

## 日报审批

组长/管理员进入 **日报 → 待审批**：

- 列出等待审批的日报
- 点击查看详情，可以：
  - 通过
  - 驳回（填写理由）

审批结果会通知提交人。

## 查看日报

- **我的日报** — 查看自己的所有日报，按月份筛选
- **团队日报** — 查看全团队的日报（需相应权限）
- **月度汇总** — 按月度统计每人工作量、类别占比

## 统计与导出

进入 **日报 → 统计**：

- 选择时间范围和人员
- 查看工作量图表、类别分布
- 导出 Excel 报表

统计可以：
- 按提交人分组
- 按类别分组
- 按关联 CMDB 资产统计

## 权限

| 权限 | 能做什么 |
|------|---------|
| `daily:create` | 填写日报 |
| `daily:approve` | 审批他人日报 |
| `daily:read_all` | 查看全团队日报 |
| `daily:export` | 导出统计 |

组级隔离：组级用户只能看本组的日报。
