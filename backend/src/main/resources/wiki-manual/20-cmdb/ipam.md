# IP 地址池管理

IPAM（IP 地址管理）用于规划和跟踪 IP 网段及其分配情况，避免 IP 冲突和地址浪费。

## 核心概念

- **IP 地址池（网段）** — 一段连续的 IP 地址范围，例如 `192.168.1.0/24`
- **IP 分配** — 把某个具体 IP 分配给某个用途/设备/配置项

## 查看地址池

进入 IPAM 页面：

- 列出所有 IP 地址池，显示网段、容量、已用/可用数量
- 点击进入查看该池内每个 IP 的分配明细

## 新建地址池

1. 点"新建地址池"
2. 填写网段（CIDR 格式，如 `10.0.0.0/24`）、名称、描述
3. 保存后系统按网段计算可用 IP 范围

## 分配与回收 IP

- **分配**：选择池内一个空闲 IP，登记用途、关联设备或配置项
- **回收**：释放已分配的 IP，使其重新可用

每个 IP 的状态一目了然：已分配 / 空闲 / 保留。

## 与 CMDB 的关系

IP 分配可关联到 [[CI 实例管理|CMDB 配置项]]，让"这个 IP 是哪台主机用的"有据可查。

## 权限

- `ip_pool:create` — 创建地址池
- `ip_pool:read` — 查看
- `ip_pool:update` — 分配/回收
- `ip_pool:delete` — 删除地址池
