# 系统配置与备份

管理员通过本模块维护平台级配置和数据备份。

## 系统配置

系统配置页可设置：

- **邮件（SMTP）配置** — 配置发件服务器，让平台能发送通知邮件（审批结果、告警等）
- **水印配置** — 页面水印的开关与内容
- 其他平台级参数

配置实时生效，修改后无需重启。

## 数据备份与恢复

备份功能对平台数据做全量备份/恢复，仅管理员可用。

### 立即备份
进入 **系统管理 → 备份**，点"立即备份"：
- 系统打包数据库 + 文件存储
- 生成一个备份文件
- 备份过程同步执行，大数据量时请耐心等待

### 上传备份
可以把本地已有的备份文件上传入库（用于迁移或恢复历史备份）。

### 下载备份
点击某条备份记录可下载备份文件，妥善保存到安全位置。

### 恢复
点"恢复"会用所选备份覆盖当前数据：

> ⚠️ 恢复是高风险操作，会覆盖现有数据。请务必确认后再操作，建议恢复前先做一次当前数据的备份。

恢复完成后建议重启服务以清除缓存。

### 自动轮转
系统自动清理超过保留期（默认 30 天）的旧备份，避免占用过多空间。

## 备份建议

- 重大变更前手动备份一次
- 定期下载备份到平台之外的安全存储
- 恢复操作前再次确认备份内容和时间点

## 权限

| 权限 | 能做什么 |
|------|---------|
| `backup:create` | 创建/上传备份 |
| `backup:read` | 查看/下载 |
| `backup:restore` | 恢复 |
| `backup:delete` | 删除备份 |

以上权限默认仅超级管理员/管理员拥有。
