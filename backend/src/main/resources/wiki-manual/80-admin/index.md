# 系统管理

系统管理面向管理员，包含用户与权限、系统配置、数据备份等平台级功能。普通用户通常无需关心本章。

## 功能导航

- **[[用户与权限管理]]** — 用户、用户组、角色、权限分配
- **[[系统配置与备份]]** — 邮件/水印等系统配置，数据备份与恢复

## 谁能访问

系统管理功能默认仅 **超级管理员** 和 **管理员** 可用。具体以角色权限分配为准。

## 权限模型概览

平台采用 RBAC（基于角色的访问控制）：

```
资源 → 权限(资源:动作) → 角色 → 用户
```

- **资源**：一类可管理对象（用户、设备、CMDB、文件等）
- **权限**：资源上的某个动作（如 `device:view_password`）
- **角色**：一组权限的集合
- **用户**：被分配一个或多个角色

用户通过角色获得权限，管理员通过调整角色权限来控制每个人能做什么。详见 [[用户与权限管理]]。
